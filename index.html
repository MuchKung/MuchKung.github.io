Cyber Security

ผู้ที่หลงใหลในเทคโนโลยีและความปลอดภัยไซเบอร์ พร้อมที่จะเรียนรู้และเติบโตในโลกแห่งความปลอดภัยดิจิทัล

ภูมิหลังและแรงบันดาลใจ

ตั้งแต่ผมจำความได้ ผมชอบเทคโนโลยีมาตลอด ไม่ว่าจะเป็นคอมพิวเตอร์ โทรศัพท์ หรืออินเทอร์เน็ต ผมชอบแกะ ชอบลอง และมักจะเป็นคนที่เพื่อนๆ มาขอความช่วยเหลือเวลาเครื่องมีปัญหา

ช่วงมัธยมปลาย ผมเริ่มรู้จักเรื่อง "Cybersecurity" จากคลิปในยูทูบ ซึ่งพูดถึงเรื่องการแฮ็ก การป้องกันข้อมูล และภัยไซเบอร์ต่างๆ ทำให้ผมรู้สึกตื่นเต้นมาก และเริ่มสงสัยว่า เบื้องหลังของความปลอดภัยในโลกออนไลน์มันทำงานยังไง

ผมคิดว่าโลกสมัยนี้พึ่งพาอินเทอร์เน็ตแทบทุกอย่าง ตั้งแต่การสื่อสาร ธุรกรรมการเงิน ไปจนถึงชีวิตประจำวัน ถ้าระบบพวกนี้โดนเจาะหรือมีปัญหา ข้อมูลสำคัญของผู้คนอาจหลุดได้ ผมจึงอยากเป็นหนึ่งในคนที่ช่วยป้องกันไม่ให้สิ่งเหล่านี้เกิดขึ้น

มุมมองและปรัชญา

ผมมองว่า Cybersecurity ไม่ใช่แค่การเขียนโค้ดหรือเทคนิคเพียงอย่างเดียว แต่มันคือการเข้าใจธรรมชาติของผู้คน เข้าใจระบบ และคิดรอบด้านเพื่อสร้างความปลอดภัยที่มั่นคง

ถึงผมจะยังไม่มีประสบการณ์มาก แต่ผมพร้อมจะเรียนรู้ ผมเริ่มศึกษาพื้นฐานพวก network, ความเข้าใจเบื้องต้นเกี่ยวกับระบบปฏิบัติการ และกำลังฝึกเขียนโค้ดบ้างจากภาษา Python กับ C ซึ่งทำให้ผมเข้าใจการทำงานของโปรแกรมมากขึ้น

ทักษะและประสบการณ์

การเขียนโปรแกรม

กำลังเรียนรู้ Python และ C เพื่อเข้าใจพื้นฐานการทำงานของโปรแกรม

ระบบเครือข่าย

ศึกษาพื้นฐานเกี่ยวกับ network และระบบปฏิบัติการ

การแก้ปัญหา

มีประสบการณ์ช่วยเพื่อนแก้ปัญหาคอมพิวเตอร์และอุปกรณ์ต่างๆ

การทำงานเป็นทีม

เคยร่วมจัดงานกิจกรรมของโรงเรียนและทำงานร่วมกับผู้อื่น

เป้าหมายและความฝัน

เป้าหมายของผมคือ อยากเข้าเรียนต่อในสาขา Cybersecurity เพราะเชื่อว่าการเรียนที่นี่จะเปิดโอกาสให้ผมได้เข้าใจทั้งภาคทฤษฎีและปฏิบัติ ได้ร่วมมือกับคนที่สนใจเหมือนกัน และสุดท้ายผมอยากเติบโตเป็นผู้เชี่ยวชาญที่สามารถช่วยปกป้องข้อมูลของคนอื่นได้อย่างจริงจัง

ผมเชื่อว่า ถ้าผมได้โอกาสเข้าเรียนในสาขานี้ ผมจะตั้งใจเรียนอย่างเต็มที่ และใช้ความรู้ที่ได้มาเพื่อทำประโยชน์ให้กับสังคมในอนาคตครับ
